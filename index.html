Notion Integration Setup

1Create Notion Integration

Go to notion.so/my-integrations

Click "New integration" ‚Üí Name it "Wavy Command Center" ‚Üí Submit

Copy the "Internal Integration Token"

2Create Notion Database

Open Notion ‚Üí Create a new page ‚Üí Type "/database" ‚Üí Select "Table - Inline"

Add these columns (properties):

  • Task (Title) - already exists
  • Project (Select) - OrderFlow, DM Bot, Client Acquisition, NFCU, Other
  • Status (Select) - Complete, In Progress, Blocked
  • Priority (Select) - High, Medium, Low
  • Time Spent (Number)
  • Outcome (Text)
  • Next Step (Text)
  • Date (Date)

3Connect Integration to Database

Click "..." menu on your database ‚Üí "Add connections" ‚Üí Select "Wavy Command Center"

4Get Database ID

Copy the database URL. It looks like:
https://notion.so/workspace/2fd68729df718055a2e9c37283d54529 The DATABASE_ID is the 32-character code after your workspace name
